## 🚀 INSTALLAZIONE

### Metodo 1: Comando Globale `wincaffe` (RACCOMANDATO)

Questo metodo installa lo script in `C:\OGD\` e crea il comando globale `wincaffe`.

#### Step 1: Download

Scarica `OGD_WinCaffe_ULTIMATE_v7.2.3.zip` e decomprimi.

#### Step 2: Esecuzione Prima Volta

```powershell
# Clicca destro su OGD_WinCaffe_ULTIMATE.ps1
# Seleziona "Esegui con PowerShell"

# OPPURE da PowerShell:
cd C:\Path\To\Script
.\OGD_WinCaffe_ULTIMATE.ps1
```

#### Step 3: Accetta Disclaimer

Leggi disclaimer e accetta digitando `S`.

#### Step 4: Check Dipendenze

Lo script verifica automaticamente:
1. PowerShell 7.5+ (installa se mancante)
2. .NET 6-11 (installa se mancanti)

Conferma installazione se richiesto.

#### Step 5: Menu Installazione

```
[1] ⚙️  INSTALLA - Aggiungi comando 'wincaffe' globale
[2] ❌ DISINSTALLA - Rimuovi comando 'wincaffe'
[3] ▶️  ESEGUI - Avvia normalmente (senza installare)
```

Scegli `[1] INSTALLA`.

#### Step 6: Completamento

Lo script:
1. Copia se stesso in `C:\OGD\OGD_WinCaffe_ULTIMATE.ps1`
2. Crea backup profilo PowerShell esistente
3. Aggiunge function `wincaffe` al profilo
4. Conferma installazione

```
✓ Script copiato: C:\OGD\OGD_WinCaffe_ULTIMATE.ps1
✓ Backup profilo: C:\Users\...\profile.ps1.ogd_backup_...
✓ Profilo PowerShell aggiornato

PROSSIMI PASSI:
1. Chiudi e riapri PowerShell
2. Digita: wincaffe
3. Lo script si aprirà ovunque tu sia!
```

#### Step 7: Uso

Chiudi e riapri PowerShell. Digita:

```powershell
wincaffe
```

Lo script si apre automaticamente con privilegi admin! 🎉

### Metodo 2: Esecuzione Diretta

Se non vuoi installare il comando globale:

```powershell
# Apri PowerShell come Amministratore
cd C:\Path\To\Script
.\OGD_WinCaffe_ULTIMATE.ps1
```

**Nota:** Dovrai navigare alla cartella ogni volta.

### Disinstallazione Comando Globale

```powershell
# Esegui script
.\OGD_WinCaffe_ULTIMATE.ps1

# Scegli [2] DISINSTALLA
```

Lo script rimuove:
- File in `C:\OGD\`
- Function `wincaffe` dal profilo PowerShell
- Cartella `C:\OGD\` se vuota

   - `bcdedit /set useplatformclock true`

   - **Beneficio:** Timer resolution 0.5ms, input lag -20%

**Sintomo:** Sistema si blocca/congela

**Causa:** RARA (fix applicato v6.5.1)

**Soluzione:**
1. Riavvia in Safe Mode (F8 boot)
2. Ripristina punto ripristino OGD
3. Se problema persiste, usa livello NORMALE invece di AGGRESSIVO

**Nota:** Fix Responsiveness 3 risolve freeze in v6.5.1+

---

#### ❌ FPS peggiorati invece di migliorati

**Sintomo:** Performance peggiori dopo ottimizzazione

**Causa:** Possibili cause multiple

**Diagnosi:**

1. **RAM insufficiente?**
   - AGGRESSIVO richiede 16GB+
   - Verifica uso RAM in Task Manager
   - Se >90%, usa livello inferiore

2. **Superfetch/Prefetcher errati?**
   - Verifica tipo RAM (DDR4 vs DDR5)
   - Menu scelta RAM corretta?

3. **Timer 0.5ms non avviato?**
   - Dopo riavvio, avvia `OGD_Timer_0.5ms.ps1` (Desktop)
   - Minimizza timer

4. **GPU drivers vecchi?**
   - Aggiorna driver GPU
   - HwScheduling richiede driver recenti

### Installazione

**Q: Devo reinstallare dopo Windows Update?**

A: **Dipende:**
- Major update (24H2→25H2): **Sì, consigliato** (alcune tweaks resettate)
- Minor update (Cumulative): **No** (tweaks preserved)
- Feature update: **Verifica**, potrebbe reinstallare bloatware

**Q: Posso disinstallare?**

A: Sì:
1. Usa punto ripristino
2. Oppure: [7] RESET dal menu
3. Oppure: [2] DISINSTALLA comando `wincaffe`

**Q: Serve riavvio?**

A: **Sì, obbligatorio:**
- Timer 0.5ms applica al boot
- C-States applica al boot
- Molte registry applica al boot

**Q: Devo avviare timer ogni volta?**

A: **Sì:**
- Timer 0.5ms NON è servizio
- Script Desktop deve essere avviato dopo ogni riavvio
- Lascia minimizzato, NON chiudere

**Futuro:** Possibile conversione a servizio Windows
